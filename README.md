# Sound Engineer Bot

Telegram бот для звукорежиссера.

Стек:

- Python 3.12
- Aiogram 3.x
- PostgreSQL
- Redis
- Docker
- Railway

## Запуск

Создать файл:

.env

на основе:

.env.example

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python main.py
```

Docker:

```bash
docker compose up --build
```
